## informações da aula
- Aula 07 do Curso de Algoritmos com Gustavo Guanabara: Estruturas Condicionais 1 (SE simples e SE..SENÃO composta).

## Pontos Principais:
- Condições são essenciais para decisões em algoritmos: "Se algo acontece, faça X".
- Estrutura simples: executa bloco se expressão for verdadeira (SE .. FimSe).
- Estrutura composta: executa bloco A se verdadeira, senão bloco B (SE .. ENTAO .. SENAO .. FimSe).
- Indentação (TAB ou Ctrl+G no Visualg) para clareza visual do código.
- Exemplos práticos: idade e maioridade (>=21), par/ímpar (n % 2 = 0), IMC ideal (18.5 <= IMC < 25).
- Exercícios: verificar habilitação (>=18 anos), aprovação escolar (média >=7).

## Resumo
Visão geral da aula: Introdução às estruturas condicionais em algoritmos usando Scratch e Visualg. Explica lógica de "se-então" com exemplos reais (viagem a Disney), demonstra sintaxe pseudocódigo, enfatiza indentação e aplica em cálculos de idade, par/ímpar e IMC. Promete faixas de IMC na próxima aula.

## comandos que aprendi
- `Se [expressão] então` ... `FimSe` (condicional simples).
- `Se [expressão] então` ... `Senão` ... `FimSe` (condicional composta).
- Operador módulo `%` (resto da divisão: n % 2 = 0 para par).
- Indentação: TAB para alinhar blocos; Ctrl+G (auto no Visualg, mas pratique manual).
