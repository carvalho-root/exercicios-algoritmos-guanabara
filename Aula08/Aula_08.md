## informações da aula
- Aula 08 do Curso de Algoritmos (Gustavo Guanabara): Estruturas Condicionais 2 (condicionais aninhadas e estrutura Escolha-Caso).

## Pontos Principais:
- Condicionais aninhadas: Usar "Se...Então...Senão" dentro de outro para múltiplas opções (ex.: aprovação de aluno: >=7 aprovado, 5-7 recuperação, <5 reprovado; classificação IMC com faixas).
- Estrutura Escolha-Caso: Para valores inteiros discretos (ex.: doação por opção 1-5; aumento salarial por nº de dependentes: 0=5%, 1-3=10%, 4-6=15%, >6=18%).
- Dicas: Indentar código, fechar cada "Se" com "FimSe"; Escolha só para inteiros exatos, não faixas.

## Resumo
Visão geral: Aula aprofunda condicionais compostas com aninhamento para cenários reais (aluno, IMC) e introduz Escolha-Caso para menus simples. Exercícios: Conceito escolar (A-F por média) e análise de jogo de futebol (empate, normal, goleada por diferença de gols).

## comandos que aprendi
- `Se (condição) Então ... Senão ... FimSe` (aninhado: um dentro do outro para ramificações).
- `Escolha (variável) Caso 1: ... Caso 2: ... Outrocaso: ... FimEscolha` (para opções exatas, inteiras).
