# Aula [05] - [Introdução ao Scratch]

## 💡 O Conceito
Nesta aula, o foco foi entender a base lógica dos algoritmos utilizando uma linguagem visual de blocos para facilitar a visualização do fluxo.

### 📚 Principais fundamentos:
- **Algoritmos:** É o caminho para a solução de um problema. Uma sequência finita de passos para atingir um objetivo.
- **Scratch:** Uma linguagem de programação visual baseada em blocos, ideal para treinar a lógica sem se preocupar com erros de digitação ou sintaxe.
- **Pense de Forma Lógica:** Antes de escrever o código, você precisa entender a sequência: Início -> Processamento -> Fim.
